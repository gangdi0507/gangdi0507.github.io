🗡️ 鏡 完整教學

王者榮耀 | 飛雷神連招詳解、最強出裝銘文、實戰打法

📌 英雄定位

  • 定位：刺客 / 打野
  • 難度：⭐⭐⭐⭐⭐（極高，需要大量練習）
  • 強勢期：4級後有大招，中期最強
  • 克制：脆皮法師、站樁射手
  • 被克制：硬控坦克（東皇、張良）、高爆發刺客（蘭陵王）
💡 核心思路：鏡的精髓在於利用大招創造鏡像空間，配合被動刷新技能，實現「飛雷神」來回穿梭秒人。會飛的鏡和不會飛的鏡，是兩個英雄。

📖 技能解析

被動 鑄鏡

  • 本體和鏡像同時擊中敵人，會造成額外傷害並刷新1、2技能CD
  • 這是鏡能「飛起來」的核心機制
  • 鏡像造成本體30%的傷害和回復

1技能 開鋒

  • 向前突進，強化下一次普攻
  • 常用作位移進場或追擊

2技能 裂空

  • 旋轉攻擊周圍敵人，命中回血
  • 清野、續航神技，配合被動可頻繁刷新

大招 見影

  • 向前飛出，定身路徑上第一個英雄，創造鏡像空間
  • 空間邊緣有8塊碎片，本體和鏡像同時攻擊碎片會造成斬殺傷害
  • 大招期間再次點擊大招，本體和鏡像互換位置，這是飛雷神的關鍵
  • 大招持續10秒，期間離開空間大招會消失

⚡ 飛雷神連招詳解

基礎飛雷神（入門）

3技能起手 → 1技能進場 → 2技能 → 平A → 3技能換位 → 1技能 → 2技能 → 平A → 3技能換位 → 重複
  • 核心節奏：本體打一套 → 換位 → 鏡像打一套 → 換位 → 循環
  • 被動刷新技能是關鍵，必須確保本體和鏡像同時命中

進階飛雷神（高手）

3技能定身 → 1技能 → 平A → 2技能 → 平A → 3換位 → 平A → 1技能 → 2技能 → 3換位
  • 在兩次換位之間穿插平A，最大化被動傷害
  • 熟練後可以在大招空間內無限穿梭

秒人連招（蹲草/抓人）

1技能突進 → 平A → 3技能定身 → 2技能 → 平A → 1技能 → 3換位追擊
  • 適用於抓單，3技能起手定身保證後續技能全中
🎯 飛雷神口訣：打一套 → 換 → 打一套 → 換 → 打一套。節奏是「12A3換 12A3換」。

⚔️ 最強出裝

  1. 追擊刀鋒（紅色打野刀，疊攻擊力）
  2. 抵抗之靴（減控，防止進場被秒控）
  3. 暗影戰斧（穿透+CD，刺客核心裝）
  4. 宗師之力（強化普攻，配合1技能爆發）
  5. 破軍（斬殺傷害，收割必備）
  6. 名刀·司命（保命，後期換復活甲）
🧠 出裝思路：純輸出，靠飛雷神規避傷害而不是扛傷害。名刀是你唯一的容錯率。

📜 銘文推薦

  • 紅色：10 異變（物理攻擊+穿透）
  • 綠色：10 鷹眼（物理穿透）
  • 藍色：10 隱匿（物理攻擊+移速）

百穿銘文，最大化前期傷害，抓人更致命。

🎮 實戰打法

前期（1-4級）

  • 紅開或藍開都可，速刷到4級
  • 4級前不要主動抓人，除非對面壓線過深
  • 鏡4級質變，有大招再考慮進攻

中期（4-12級）

  • 這是鏡最強勢的時期，多抓發育路射手
  • 蹲草用大招起手，一套秒脆皮
  • 小規模團戰先手定身對面C位

後期（12級後）

  • 不要先手進場！等隊友開團
  • 從側面切入，找對面射手/法師
  • 名刀是你的底牌，沒名刀不要浪

💎 進階技巧

  • 大招穿牆：大招的飛行可以穿過薄牆，用來逃生或追擊
  • 無傷打龍：開大招打龍，用飛雷神來回換位，龍的攻擊打不到你
  • 假換位：有時不換位比換位更讓對面混亂，騙出技能後再換
  • 草叢大招：躲在草裡開大，對面看不到鏡像空間，進來的瞬間就被定身
