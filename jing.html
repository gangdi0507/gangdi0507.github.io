🗡️ 镜完整教学

王者荣耀 | 飞雷神连招详解、最强出装铭文、实战打法

📌 英雄定位

  • 定位：刺客 / 打野
  • 难度：⭐⭐⭐⭐⭐
  • 强势期：4级后有大招
  • 克制：脆皮法师、站桩射手
  • 被克制：东皇、张良等硬控
💡 核心思路：镜的精髓在于利用大招创造镜像空间，配合被动刷新技能，实现「飞雷神」来回穿梭秒人。

📖 技能解析

被动 铸镜

  • 本体和镜像同时命中敌人，造成额外伤害并刷新1、2技能CD

1技能 开锋

  • 向前突进，强化下一次普攻

2技能 裂空

  • 旋转攻击周围敌人，命中回血

大招 见影

  • 向前飞出，定身第一个英雄，创造镜像空间
  • 大招期间再次点击，本体和镜像互换位置

⚡ 飞雷神连招

基础飞雷神

3技能起手 → 1技能进场 → 2技能 → 平A → 3换位 → 1技能 → 2技能 → 平A → 3换位 → 重复

秒人连招

1技能突进 → 平A → 3技能定身 → 2技能 → 平A → 1技能 → 3换位追击
🎯 口诀：打一套 → 换 → 打一套 → 换 → 打一套。节奏是「12A3换 12A3换」。

⚔️ 推荐出装

  1. 追击刀锋
  2. 抵抗之靴
  3. 暗影战斧
  4. 宗师之力
  5. 破军
  6. 名刀·司命
