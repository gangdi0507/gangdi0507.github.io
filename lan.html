🦈 瀾完整教學

王者榮耀 | 鯊魚王打野連招、出裝銘文、抓人節奏

📌 英雄定位

  • 定位：刺客 / 打野
  • 難度：⭐⭐⭐（中等，連招需練習）
  • 強勢期：4級後有大招，抓人能力質變
  • 克制：脆皮射手、法師
  • 被克制：硬控（東皇、張良）、高爆發刺客
💡 核心思路：瀾的優勢在於超強機動性和收割能力。利用河道加速快速支援，大招推人回來配合隊友擊殺。殘血敵人在瀾面前就是獵物。

📖 技能解析

被動 狩獵

  • 敵人生命低於30%時，瀾對其造成額外20%真實傷害
  • 殘血收割機的核心機制

1技能 破浪

  • 潛入水中獲得加速，持續3秒
  • 河道中無時間限制，可以一直潛水
  • 出水時向前跳躍，造成傷害

2技能 斷空

  • 旋轉攻擊周圍敵人兩次，標記目標
  • 普攻被標記的敵人會突進過去
  • 是瀾的主要輸出技能

大招 處決

  • 向前扔出匕首眩暈敵人，然後突進到目標身後
  • 可以把敵人推回來，配合隊友擊殺
  • 核心抓人技能

⚡ 連招教學

基礎抓人連招

  • 1技能潛水接近 → 出水跳躍 → 2技能旋轉標記 → 普攻突進 → 大招推回來

進階收割連招

  • 1技能潛水 → 大招起手推人 → 2技能旋轉 → 普攻突進追擊 → 1技能再潛水追擊
🧠 口訣：「河道潛水無限跑，見到殘血直接咬。大招推回來隊友秒，被動真傷收尾刀。」

⚔️ 推薦出裝

  1. 追擊刀鋒（紅色打野刀）
  2. 抵抗之靴（減控）
  3. 暗影戰斧（穿透+CD）
  4. 宗師之力（強化普攻）
  5. 破軍（配合被動收割更強）
  6. 名刀·司命（保命換復活甲）

📜 銘文推薦

  • 紅色：10 異變
  • 綠色：10 鷹眼
  • 藍色：10 隱匿

🎮 實戰打法

  • 前期：紅開或藍開，速刷到4級，不要浪費時間蹲人
  • 中期：利用河道加速頻繁遊走，優先抓發育路
  • 後期：團戰不要先手，等隊友開團後從側面切入收割殘血
  • 被動對殘血有額外傷害，團戰先打前排也可以，等對面C位殘了再追
