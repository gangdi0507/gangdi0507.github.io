🗡️ 馬超完整教學

王者榮耀 | 插秧王連招、出裝銘文、實戰打法

📌 英雄定位

  • 定位：刺客 / 戰士
  • 難度：⭐⭐⭐⭐（需要練習）
  • 強勢期：4級後有大招，中期最強
  • 克制：脆皮法師、站樁射手
  • 被克制：硬控坦克（東皇、張良）、高爆發刺客
💡 核心思路：馬超靠著扔槍、撿槍、強化普攻的循環打出高額傷害。會玩的馬超全圖亂飛，不會玩的馬超全圖亂送。

📖 技能解析

被動 魔影突襲

  • 每次釋放技能增加移速，可疊加3層
  • 撿起標槍後，下一次普攻變為強化穿刺，造成額外傷害並回復生命
  • 強化普攻命中敵人減所有技能1秒CD

1技能 蕭索之刃

  • 向指定方向扔出標槍，造成傷害
  • 標槍落在地上，需要撿起來才能觸發強化普攻

2技能 日落孤槍

  • 向指定位置插下標槍，造成範圍傷害和減速
  • 同樣需要撿起來

大招 萬刃歸鞘

  • 召回最近的標槍，標槍飛回過程中對路徑敵人造成傷害
  • 可連續召回最多4把標槍
  • 每次召回都附帶強化普攻效果

⚡ 連招教學

基礎連招（插秧）

  • 1技能扔槍 → 2技能插槍 → 走過去撿 → 強化普攻戳出去
  • 核心節奏：扔槍 → 撿槍 → 戳 → 扔槍 → 撿槍 → 戳

大招連招

  • 1技能扔槍 → 2技能插槍 → 3技能召回 → 強化普攻戳 → 3技能再召回 → 再戳
  • 大招期間最多戳4次，傷害爆炸
🧠 口訣：「扔槍插槍，撿起來戳。大招一開，來回戳四次。」

⚔️ 推薦出裝

  1. 冷靜之靴（減CD，技能轉更快）
  2. 暗影戰斧（穿透+CD）
  3. 宗師之力（強化普攻傷害）
  4. 破軍（斬殺傷害）
  5. 不祥徵兆（增加生存）
  6. 名刀·司命（保命）

📜 銘文推薦

  • 紅色：10 異變（物理攻擊+穿透）
  • 綠色：10 鷹眼（物理穿透）
  • 藍色：10 隱匿（物理攻擊+移速）

🎮 實戰打法

  • 前期：4級前弱勢，猥瑣發育，用1、2技能消耗
  • 中期：4級後有大招開始遊走抓人，優先抓發育路
  • 後期：從側面進場，先扔槍再開大招收割
  • 團戰不要第一個進場，等隊友開團後再切入
